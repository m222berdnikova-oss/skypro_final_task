Git

Основные команды Git

  • git init – Инициализация нового репозитория в текущей папке
  • git add – Добавление файлов в область подготовленных изменений (staging area)
  • git commit -m "сообщение" – Сохранение изменений в репозитории с комментарием
  • git status – Проверка состояния репозитория (какие файлы изменены или готовы к коммиту)
  • git push – Отправка изменений на удалённый репозиторий (например, GitHub)
  • git pull – Получение изменений из удалённого репозитория
  • git clone – Копирование удалённого репозитория на локальный компьютер
  • git log – Просмотр истории коммитов
